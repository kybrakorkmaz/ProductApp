# Product App

A backend application developed in Go, based on a Udemy course and further extended with custom improvements.

The backend was refactored to migrate from **PostgreSQL v4 to v5**.  
**DataGrip** was used for database management and data visualization during development.

The frontend will be developed using **React**.

### Tech Stack & Tools
- Go
- Echo
- PostgreSQL
- React (planned)
- GoLand (IDE)
- DataGrip
- Postman (API testing)

### Testing
Controller endpoints were tested manually using **Postman**.

### Planned Enhancements
- Centralized and well-structured error handling
- Frontend integration with React
